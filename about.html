about this project, and myself

why this project?

The goal of this project was to create an Android application that would display hymns. These hymns were copyrighted and printed in a book, titled Hymns, published by the Living Stream Ministry, and were generously put online at hymnal.net. There is also a mobile download already available, a .epub file that can be read by the iSilo reader.

I wanted a project to work on during my first semester at UC Berkeley, and I wanted to experiment with scraping websites and displaying them on client screens like smartphones compactly. I imagined that this could be applied to other websites as well, such as Ultimate Guitar. My reasoning was that a website should not have a free, ad-supported site which can be viewed by a mobile device any time, but force user to buy an app to use it more conveniently.

Though I've now realized that there's no way that a website-scraping app that displays the content of webpages would never fly with copyright rules, I still learned a lot from this application's development.
